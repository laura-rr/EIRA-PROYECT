## 3. EIRA - SISTEMA GESTION DE PRESTAMOS INTELIGENTES
Descripcion:
EIRA es un sistema diseñado para gestionar de manera eficiente el inventario de objetos y los préstamos realizados entre usuarios. Permite llevar un control organizado, automatizar procesos como préstamos, devoluciones, notificaciones y generación de documentos, ofreciendo una interfaz intuitiva y fácil de usar para cualquier usuario.

## 5. REPORTE DE VISION
El sistema busca facilitar la gestion de prestamos de objetos, permitiendo llevar un control claro ddel inventario y los usuarios.

**Objetivos:**
- Organizar el Inventario
- Controlar Prestamos
- Generar Alertas
- Facilitar el acceso a informacion

**Beneficios:**
- Evita perdida de objetos
- Mejora el control
- Automatiza procesos
- Facilidad de Uso

## 6. ESPECIFICACIONES DE REQUISITO
**El sistema debe permitir:**
- Registrar usuarios con validaciones (nombre, documento, correo, etc.).
- Registrar ítems con categoría, precio, ID único y estado.
- Crear préstamos únicamente a usuarios registrados.
- Registrar devoluciones y generar certificados.
- Generar ventas automáticas después de 30 días.
- Calcular el impuesto del 23% en ventas.
- Consultar el estado general de préstamos.
- Acceder a un módulo administrador con usuario y contraseña.
- Generar reportes:
  - Total de préstamos
  - Total de devoluciones
  - Total de ventas
  - Total recaudado
  - Lista de usuarios
  - Usuario con mayor y menor cantidad de préstamos

   **REQUISITOS NO FUNCIONALES**
  - Usabilidad: Interfaz sencilla e intuitiva.
  - Rendimiento: Respuesta rápida en consultas y registros.
  - Seguridad: Validación de acceso para el administrador.
  - Fiabilidad: Almacenamiento correcto de datos en archivos.
  - Compatibilidad: Ejecución en cualquier sistema con Python.
  - Mantenibilidad: Código organizado y estructurado.
